## Scheduled-Task

There are no permanent services, all the programs run on demand, most of which is via a Windows Scheduled Task either on [[RIVSIS02]] or [[RIVSME01]].

The batch files can [viewed in the repo](https://dw.ramsden-international.com/gogs/matthew.heath/RocsMiddleware/src/master/RI-Services-Scripts).

The binaries are located in \\[[RIVSTS05]]\Software\RocsMiddleware
